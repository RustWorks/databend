---
title: zeroUUID
---

Generate zero UUID.

## Syntax

```sql
zeroUUID()
```

## Return Type

Zero UUID.

## Examples

```sql
mysql> select zeroUUID();
+--------------------------------------+
| zeroUUID()                           |
+--------------------------------------+
| 00000000-0000-0000-0000-000000000000 |
+--------------------------------------+
```
