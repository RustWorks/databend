---
title: generateUUIDv4
---

Generate a random UUID based on v4.

## Syntax

```sql
generateUUIDv4()
```

## Return Type

Random UUID.

## Examples

```sql
mysql> select generateUUIDv4();
+--------------------------------------+
| generateUUIDv4()                     |
+--------------------------------------+
| 0ec1cbb8-3d6f-4e85-ac65-046c75721b76 |
+--------------------------------------+
```
