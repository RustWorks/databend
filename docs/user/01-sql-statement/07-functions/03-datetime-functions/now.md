---
title: NOW
---

Returns the current date and time.

## Syntax

```sql
NOW()
```

## Return Type

Datetime object, returns date and time in “YYYY-MM-DD HH:MM:DD” format.

## Examples

```sql
mysql> select NOW();
+---------------------+
| NOW()               |
+---------------------+
| 2021-09-03 01:26:07 |
+---------------------+
```
