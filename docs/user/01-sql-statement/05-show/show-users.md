---
title: SHOW USERS
---

Shows the list of user accounts.

## Syntax

```
SHOW USERS
```

## Examples

```sql
mysql> SHOW USERS;
+------+-----------+----------+-----------+
| name | hostname  | password | auth_type |
+------+-----------+----------+-----------+
| test | localhost | password |         3 |
+------+-----------+----------+-----------+
```
